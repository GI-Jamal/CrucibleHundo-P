The Code

                    
//get the start and end numbers from the inputs
// Entry point AKA Controller function
function getValue() {
  let startValue = document.getElementById("startValue").value; // startValue = "0"
  let endValue = document.getElementById("endValue").value;

  let startNumber = parseInt(startValue); // startNumber = 0
  let endNumber = parseInt(endValue);

  if (isNaN(startNumber) || isNaN(endNumber))
  {
    Swal.fire(
      {
        icon: 'error',
        title: 'Oops!',
        text: 'Please enter valid numbers for the start and end values',
        heightAuto: 'false'
      }
    );
  } 
  
  else
  {
    let numbers = generateNumbers(startNumber, endNumber);

    displayNumbers(numbers);
  }
}

// generate the range of numbers to display
// Business/logic function
function generateNumbers(start, end) {
  let numbers = [];

  for (let i = start; i <= end; i++) {
    numbers.push(i);
  }

  return numbers;
}

// placing the generated numbers on the page AND bolding the even numbers
// View function
function displayNumbers(theNumbers) {
  
  let results = '';
  
  for (let i = 0; i < theNumbers.length; i++) {
    let currentNumber = theNumbers[i];

    if (currentNumber % 2 == 0)
    {
      results = results += `<tr><td class="evenNumber">${currentNumber}</td></tr>`;
    }
    else
    {
      results += `<tr><td>${currentNumber}</td></tr>`;
    }
  }

  let tableBody = document.getElementById('results');
  tableBody.innerHTML = results;
}

The code is structured in three functions

getValue():

The getValue function retrieves the values from the users input and stores them in two variables named startValue and endValue for the starting number and ending number respectively.

It then parses the input from text into numeric input and stores those numbers in two new variables before calling the generateNumbers function and passing it those new numeric variables.

The results from the generateNumbers function is stored in a variable called numbers.The displayNumbers function is then called and the variable numbers is passed into it.

generateNumbers(start, end):

The generateNumbers function takes two parameters, a start and an end value, and creates an array of numbers from the first number to the second number, inclusive.

It does this by using a for loop to iterate through each number to add it to the array. It then returns this filled array to the caller.

displayNumbers(theNumbers):

The displayNumbers function take a single parameter of an array of numbers and displays the numbers in a table with the even numbers displayed in bold and red. It does this by manipulating the innerHTML property of the document element that has an Id of results.

In this case it is a tbody tag. The function creates a variable to store the HTML string for the numbers, and then iterates through the array of numbers, determining if a number is even by using the modulus operator, then adds string of HTML text to the results variable, with even numbers having a CSS class of evenNumbers added to them.

This results variable is then assigned to the innerHTML property of the tbody tag display the numbers in a tabular format.
